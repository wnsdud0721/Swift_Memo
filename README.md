# Swift_Memo

### ✅ 프로젝트 개요

- 프로젝트 명: Memo
- 목표 : Swift, UIKit을 이용하여 메모장 앱 구현
- 필수 작업 : Navigation Controller 사용, 스토리보드/인터페이스 빌더를 사용, UITableView 사용
- 추가 작업 : 데이터 처리를 통해 메모 작성 및 저장, 삭제 기능

### ✅ Problem

1) 문제
- Navigation Bar 구현
- 메모를 작성하지 않아도, 메모 리스트에 추가
- 앱을 실행시켜도 badge에 숫자가 사라지지 않음

2) 원인
- iOS 환경에서 제공하고 있는 기능에 대한 정보 부족
- Xcode 내의 기능 파악 부족
- 구현 기능 원리 파악 부족
- 함수가 호출이 되지 않음
- AppDelegate와 SceneDelegate에 대한 지식 부족

3) 해결
- **Large Title** 기능 이용
- Text View에 place holder 처럼 보이게 하는 부분을 분석
- **textViewDidChange** 메소드 이용
- AppDelegate가 아닌 SceneDelegate에서 badge 숫자 설정
